# TP Système de Santé

##  Description
Ce projet contient les trois parties du TP de modélisation UML et développement Python.

##  Structure du Projet

### Partie A : Modélisation UML d'un système de bibliothèque
- Diagramme de classes UML
- Implémentation Python

### Partie B : Système bancaire
- Diagramme de classes UML  
- Implémentation Python (Client/Compte)

### Partie C : Système de paiement en ligne (SPL)
- Diagramme de cas d'utilisation
- Diagramme de classes
- Implémentation Python complète
